# File Size Checker

A simple Python tool that scans folders on macOS and Windows to calculate total file size and number of files. It also includes a Trash Bin manager.

## Features
- Scan Desktop, Downloads, and Documents
- Shows total size in KB and MB
- Counts total files
- View Trash Bin size
- Option to empty Trash Bin safely
- Repeat menu option for continuous use
